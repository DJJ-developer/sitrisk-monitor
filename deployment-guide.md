## 📋 **1단계: GitHub Pages 배포** (15분)

### 🌐 **GitHub 저장소 생성**
```bash
# 1. GitHub.com 로그인
# 2. 우상단 '+' 버튼 클릭 > New repository
# 3. 저장소 정보 입력:
Repository name: sitting-health-monitor
Description: 실시간 앉기 위험도 측정기 - SitRisk Monitor
Public 선택
```

### 📁 **파일 업로드**
```bash
# 방법 1: GitHub 웹 인터페이스
# - 저장소 페이지에서 "uploading an existing file" 클릭
# - 모든 파일을 드래그앤드롭

# 방법 2: Git 명령어 (추천)
git init
git add .
git commit -m "🚀 SitRisk Monitor 초기 배포"
git remote add origin https://github.com/[사용자명]/sitting-health-monitor.git
git push -u origin main
```

### ⚙️ **GitHub Pages 활성화**
```
1. 저장소 > Settings 탭 클릭
2. 왼쪽 메뉴에서 "Pages" 클릭
3. Source: "Deploy from a branch" 선택
4. Branch: "main" 선택
5. "Save" 버튼 클릭
6. ✅ 5-10분 후 사이트 활성화
```

### 🔗 **AdSense 계정 생성**
```
1. https://www.google.com/adsense/ 방문
2. "시작하기" 버튼 클릭
3. Google 계정으로 로그인
4. 웹사이트 정보 입력:
   - URL: https://[사용자명].github.io/sitting-health-monitor/
   - 국가: 대한민국
   - 결제 통화: KRW (한국 원)
```

### 📊 **사이트 연결**
```
5. "사이트를 AdSense에 연결" 선택
6. AdSense 코드 복사
7. <head> 태그에 코드 추가 (이미 준비됨)
8. "사이트에 코드를 배치했습니다" 체크
9. "완료" 클릭
```

## 🔧 **3단계: 승인 후 설정** (20분)

### 🆔 **Publisher ID 교체**
```javascript
// AdSense 승인 후 받은 실제 Publisher ID로 교체

// monitor.html 파일에서:
ca-pub-XXXXXXXXXX → ca-pub-1234567890123456

// index.html 파일에서도 동일하게 교체
// (실제 ID는 AdSense 대시보드에서 확인)
```

### 📊 **광고 단위 생성**
```
AdSense 대시보드에서:
1. "광고" > "광고 단위별" 클릭
2. "디스플레이 광고" 선택
3. 광고 단위 설정:
   - 이름: SitRisk_Header_Banner
   - 크기: 반응형
   - 광고 유형: 텍스트 및 디스플레이 광고
4. "만들기" 클릭
5. 생성된 코드를 상단/하단 배너에 적용
```

### ⚙️ **자동 광고 설정**
```
1. "광고" > "개요" 클릭
2. "자동 광고" 토글 켜기
3. 광고 형식 선택:
   - [✓] 페이지 내 광고
   - [✓] 앵커 광고 (모바일)
   - [✓] 사이드 레일 광고 (데스크톱)
   - [○] 수익 창출 최적화 (테스트 후 적용)
```
